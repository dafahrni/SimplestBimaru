# Just another Bimaru

## Setup

- install [node.js](https://nodejs.org/en/download)
- execute `npm install`


## Available Build Commands

- `npm run clean`
- `npm run build`
- `npm run release`


## Launch Scripts

- `npm test`
- `npm start`


## Debugging in VS Code

- go to marketplace and install *Live Server* extension
- right mouse click on `index.html` to open with *Live Server*
